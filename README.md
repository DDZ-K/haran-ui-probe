# HARAN UI 探测 v2.1 · 自定义区域 + 多模板

## 为什么要 v2.1

视频显示「空闲」时底栏**不是固定一张图**，会在例如：

- `State: Currently no Repair Data`
- `State: archiving Result File`
- 其它短暂状态

之间切换。所以：

1. **空闲要录多张模板**（每种闪一下的外观各存一张）  
2. **截取范围要可自定义**（框选或手填 L/T/W/H）

## 使用

1. 打开 HARAN，管理员则本工具也管理员  
2. 点 **框选区域…** 拖出状态栏（或更大）搜索框，或手调参数  
3. **空闲**：等底栏变一次就 **加入空闲模板** 一次（至少 2 张）  
4. **待判 Waiting**：再 **加入待判模板**（1～2 张）  
5. **立即匹配** / **自动轮询**  

| 颜色 | 含义 |
|------|------|
| 绿 | 命中待判模板 → 可判定 |
| 蓝 | 命中任一空闲模板 |
| 橙 | 未知（补模板或调区域/阈值） |

模板目录：

```text
templates/idle/*.png
templates/waiting/*.png
templates/roi.txt
```

## 参数

- **相对底边定位**：从窗口底边往上截 H 像素  
- **框选**：自动改为绝对 L/T/W/H  
- **W=0**：铺满到窗口右边缘  
- **阈值**：默认 0.86；误报升高、漏检降低
